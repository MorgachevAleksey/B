Переменные

По умолчанию - переменные в rust неизменяемы
Обьявляются с помощью операторf let
Можно сделать изменяемыми с помощью mut

Константы

В отличие от переменных, их нельзя сделать изменяемыми
Обьявляются с помощью const
Обьявляются в любой области видимости, включая глобальную
Обьявляются только явно - не могут быть результатом вычислений
!Тип значения обязательно должен быть указан
Зачастую пишутся upper_case
